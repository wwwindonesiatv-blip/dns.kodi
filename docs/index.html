📥 Quick Install

📦 Download Repository 📖 Install Guide

Installation Methods

Method 1: File Manager (Recommended)

  1. Kodi → Settings → File Manager → Add source
  2. URL: https://wwwindonesiatv-blip.github.io/dns.kodi/
  3. Name: DNS Kodi Repo
  4. Add-ons → Install from zip → DNS Kodi Repo → zips → repository.dns.kodi

Method 2: Direct Download

📦 Download repository.dns.kodi.zip

After Installation

  1. Add-ons → Install from repository
  2. DNS Kodi Repository → Services
  3. Google DNS Auto → Install
  4. Restart Kodi → DNS auto-changes! ✅
