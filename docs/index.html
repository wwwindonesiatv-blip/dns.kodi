Google DNS Repo

Alamat sumber resmi untuk plugin Auto-Switch Google DNS Kodi.

https://wwwindonesiatv-blip.github.io/dns.kodi/zips/
Panduan Cepat:
  1. Settings > File Manager > Add Source.
  2. Masukkan URL hijau di atas, beri nama .DNS.
  3. Add-ons > Install from zip file > .DNS.
  4. Pilih repository.dns.google-x.x.x.zip.
