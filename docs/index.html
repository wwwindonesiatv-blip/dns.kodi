Google DNS Repository

Salin URL ini ke File Manager Kodi:

https://wwwindonesiatv-blip.github.io/dns.kodi/zips/
Cara Pasang:
  1. Settings > File Manager > Add Source.
  2. Masukkan URL di atas, beri nama .DNS.
  3. Add-ons > Install from zip file > pilih .DNS.
  4. Klik repository.dns.google-1.0.0.zip.
  5. Install from repository > Google DNS Repo > Services > Google DNS Auto.
