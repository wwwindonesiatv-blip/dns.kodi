Google DNS Repository

Official repository for Google DNS Auto-Switch Addon

https://wwwindonesiatv-blip.github.io/dns.kodi/zips/

Cara Install di Kodi:

  1. Buka Kodi, masuk ke Settings (ikon gear).
  2. Pilih File Manager > Add Source.
  3. Klik <None> dan masukkan URL di atas tepat seperti yang tertulis.
  4. Beri nama sumber ini, misalnya: .GoogleDNS
  5. Kembali ke Home Screen, pilih Add-ons.
  6. Klik ikon Package Installer (box terbuka) di pojok kiri atas.
  7. Pilih Install from zip file.
  8. Cari nama .GoogleDNS dan pilih file repository.dns.google-1.0.0.zip.
  9. Setelah terinstall, pilih Install from repository > Google DNS Repository.
