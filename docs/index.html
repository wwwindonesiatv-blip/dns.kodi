🌐 DNS Kodi Repository

Auto DNS Changer for Android Devices

📥 Quick Install

Method 1: File Manager Source

  1. Open Kodi → Settings → File Manager
  2. Click Add source
  3. URL: https://USERNAME.github.io/dns.kodi/
  4. Name: DNS Kodi Repo
  5. Click OK
  6. Go to Add-ons → Install from zip file
  7. Select DNS Kodi Repo → zips → repository.dns.kodi
  8. Install repository.dns.kodi-1.0.0.zip
  9. Wait for notification: "Repository enabled"

Method 2: Direct Download

  1. Download repository ZIP: 📦 repository.dns.kodi-1.0.0.zip
  2. Open Kodi → Add-ons → Install from zip file
  3. Navigate to downloaded ZIP
  4. Install and wait for confirmation

Method 3: Command Line (Advanced)

Copy dan paste ke Kodi Add-on Browser URL:

https://USERNAME.github.io/dns.kodi/zips/repository.dns.kodi/repository.dns.kodi-1.0.0.zip

📦 Install Addon from Repository

  1. Add-ons → Install from repository
  2. Select DNS Kodi Repository
  3. Services → Google DNS Auto
  4. Click Install
  5. Restart Kodi → DNS automatically changes!
