📥 Quick Install

Method 1: One-Click Install (Recommended)

  1. Download repository ZIP: 📦 repository.dns.kodi-1.0.0.zip
  2. Di Kodi: Add-ons → Install from zip file
  3. Pilih file ZIP yang didownload
  4. Tunggu notifikasi "Repository enabled"
  5. Install addon: Add-ons → Install from repository → DNS Kodi Repository → Services

Method 2: File Manager (Untuk Firestick)

  1. Di Kodi: Settings → File Manager → Add source
  2. URL: https://raw.githubusercontent.com/wwwindonesiatv-blip/dns.kodi/main/
  3. Name: DNS Kodi Repo
  4. Install: Add-ons → Install from zip → DNS Kodi Repo → zips → repository.dns.kodi

❓ FAQ

Apakah perlu root?

Tidak! Addon ini pakai Android Private DNS API (Android 9+)

Cara cek DNS sudah berubah?

Buka browser → dnsleaktest.com → Harus muncul "Google LLC"

Cara disable addon?

Add-ons → My add-ons → Services → Google DNS Auto → Disable
