📖 Installation Guide

Panduan lengkap install DNS Kodi Repository di semua device Android

1 Download Repository ZIP

Download file repository dari link ini:

📦 repository.dns.kodi-1.0.0.zip

Simpan di:

  • Firestick: Download folder
  • Android Box: Internal storage
  • PC: Desktop

2 Enable Unknown Sources

Di Kodi:

  1. Settings → System → Add-ons
  2. Aktifkan Unknown sources
  3. Klik Yes pada warning

3 Install dari ZIP

Di Kodi:

  1. Add-ons → Install from zip file
  2. Navigasi ke lokasi file ZIP
  3. Pilih repository.dns.kodi-1.0.0.zip
  4. Tunggu notifikasi "DNS Kodi Repository Add-on enabled"

4 Install Addon

Di Kodi:

  1. Add-ons → Install from repository
  2. Pilih DNS Kodi Repository
  3. Services → Google DNS Auto
  4. Klik Install
  5. Tunggu sampai selesai

5 Restart Kodi

Restart Kodi untuk aktivasi service:

  1. Close Kodi completely
  2. Buka Kodi lagi
  3. Tunggu 5 detik
  4. Notifikasi muncul: "DNS berhasil diubah ke Google DNS"

6 Verifikasi DNS

Cek apakah DNS sudah berubah:

Via ADB (Command Line):

adb shell settings get global private_dns_specifier
# Output: dns.google

Via Browser Android:

  1. Buka browser → dnsleaktest.com
  2. Klik Extended test
  3. Harusnya muncul Google LLC

✅ Selesai!

Setiap kali Kodi dibuka, DNS otomatis berubah ke Google DNS (8.8.8.8).

Tidak perlu setting lagi!

🔧 Troubleshooting

Notifikasi "DNS berhasil diubah" tidak muncul?

  1. Cek System → Logging untuk error
  2. Pastikan device Android 9+
  3. Restart Kodi 2-3 kali

Error "Invalid structure" saat install?

  1. Download ulang ZIP file
  2. Pastikan ZIP tidak corrupt
  3. Jangan extract ZIP, install langsung

Cara uninstall addon?

  1. Add-ons → My add-ons → Services
  2. Google DNS Auto → Uninstall
  3. DNS kembali ke setting ISP
